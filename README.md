# PawMart Server - Backend API

## Features

# RESTful API Endpoints - Provides comprehensive API routes for managing categories, listings, and orders with proper error handling and response formatting.

# MongoDB Integration - Seamlessly connected to MongoDB Atlas for efficient data storage and retrieval of pet listings, orders, and category information.

# CRUD Operations for Listings - Complete create, read, update, and delete functionality for pet and product listings with ObjectId-based routing.

# Order Management System - Handles order creation and retrieval with support for tracking all adoption requests and product purchases.

# Secure Environment Configuration - Uses dotenv for secure credential management and implements CORS for safe cross-origin resource sharing between client and server.
